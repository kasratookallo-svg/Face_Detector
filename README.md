# 🎯 Face Detector (Real-Time)

A simple and efficient real-time face detection system using OpenCV.  
This project captures video from a webcam and detects human faces using classical computer vision techniques.

---

## 📌 Features

- 🎥 Real-time webcam face detection  
- ⚡ Fast and lightweight (OpenCV-based)  
- 🧠 Uses Haar Cascade classifier  
- 📦 Easy to run and modify  
- 🖼️ Draws bounding boxes around detected faces  

---

## 🛠️ Technologies Used

- Python 3.x  
- OpenCV (cv2)  
- Haar Cascade Classifier  

---

## 📂 Project Structure

Face_Detector/
│── main.py  
│── README.md  

---

## 🚀 How It Works

1. Webcam is initialized using OpenCV  
2. Each frame is captured in real-time  
3. Frame is converted to grayscale  
4. Haar Cascade detects faces  
5. Bounding boxes are drawn on detected faces  
6. Output is displayed live
